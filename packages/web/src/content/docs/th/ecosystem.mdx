:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ AWMate ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-AWMate](https://github.com/awesome-AWMate/awesome-AWMate) และ [AWMate.cafe](https://awmate.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
